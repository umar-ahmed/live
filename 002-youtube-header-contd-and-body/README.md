# Recreating YouTube Home Page Part 2: Improving Header and Starting on Body

In this session, we refactor our HTML and CSS into separate files and learn a little bit about SVG. We also start working on the topics list section of the Body, discussing advanced CSS patterns like the "lobotomized owl selector" and the `:not()` function in CSS.
